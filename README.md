# Learning_Dart
Script per imparare il linguaggio di programmazione Dart

Eseguire gli script con il comando `dart run nome_script.dart`

## numbers.dart
Utilizzo dei numeri:
- Numeri interi `int` e reali `dobule`
- Parsing di un numero intero/reale da una stringa

## strings.dart
Utilizzo delle stringhe:
- Concatenazione
- Substring
- Lunghezza
- Indicizzazione
